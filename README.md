## :star2: About the Project
<p>APAC is a platformer about... well, collectables. Complete levels by getting all the collectables, but try not get too distracted with the collection effect!</p>

<!-- Features -->
### :dart: Features

- >Amazing collectable animations.
- >0 art assets (Except for Saph. We love Saph here.). Solely creative thinking and amazing use of Godot's features.
- >Every jump matters more than the last.
